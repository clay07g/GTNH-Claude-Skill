## Main Progression Tiers (Linear Path)

These quest lines represent the core tech progression. Content in each tier builds upon the previous.

### And So, It Begins (display_order: 1, ~45 quests)
**Content:** Tutorial, first night survival, basic food/shelter
**Key Topics:** Punching trees, basic food (berries, potatoes), crafting table, furnace, first bed, flint crafting, animal killing mechanics (GT knife), basic mob dangers
**When to Search:** Questions about very first steps, "how do I start", food basics, animal explosions

### Tips and Tricks (display_order: 2, ~32 quests)
**Content:** Reference/guide quests, cross-tier tips
**Key Topics:** NEI usage, ore finding (3-chunk grid, F9 overlay), programmed circuits, pipe mechanics (GT6-style), EBF tips, bee basics, electricity safety, keybinds, pollution
**When to Search:** "How do I find ores", NEI help, general mechanics questions, "tips for X"

### Tier 0 - Stone Age (display_order: 3, ~92 quests)
**Content:** Pre-steam progression, Tinker's Construct, first machines
**Key Topics:** Smeltery, Tinker's tools/weapons, coke oven, charcoal, creosote, bronze crafting, ore veins (copper, tin, iron), ore finder wand, barrels/drawers, IC2 crops, horses
**When to Search:** Smeltery questions, coke oven, charcoal, early ore processing, Tinker's tools, bronze age

### Tier 0.5 - Steam (display_order: 4, ~113 quests)
**Content:** Steam machines, transition to electricity
**Key Topics:** Steam machines (macerator, compressor, extractor, forge hammer), bronze/steel boilers, solar boilers, steam turbine, basic circuits, rubber, cables, crossbow, Iron Tank, Forestry intro
**When to Search:** Steam machine questions, boilers, first circuits, rubber production, "how to make steel"

### Tier 1 - LV (display_order: 5, ~152 quests)

## Query Strategy Notes

1. **Tier-specific questions**: Query the appropriate tier quest line directly
2. **"How do I make X"**: Check tier quest lines first, then parallel lines
3. **Mechanics questions**: Check Tips and Tricks first
4. **Power questions**: Check How to Generate Power + relevant tier
5. **Processing chains**: Check the relevant tier + Bio for the Masses or Working with Oil
6. **Magic questions**: Check appropriate magic quest line
7. **Multiblock questions**: Check Multiblock Goals + Mass Processing + relevant tier
8. **Early game (pre-LV)**: And So It Begins, Tier 0, Tier 0.5
9. **Mid game (LV-HV)**: Tier 1-3, parallel quest lines unlock
10. **Late game (EV+)**: Tier 4+, AE2, nuclear, space, advanced magic
